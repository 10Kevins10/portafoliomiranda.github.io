Semana 2: HTML

Definiciones

HTML es un lenguaje de marcado que se utiliza para estructurar el contenido de una página web. Consiste en una serie de elementos o etiquetas que rodean el contenido y le dan significado.

Las etiquetas HTML son códigos encerrados entre corchetes angulares <>, y generalmente vienen en pares de apertura <> y cierre.

Para qué sirve HTML:

  • Estructurar la Información: HTML se utiliza para organizar el contenido de una página web en diferentes secciones, como encabezados, párrafos, listas y tablas, lo que facilita la comprensión y presentación de la información.
  • Creación de Hipervínculos: HTML permite la creación de enlaces o hipervínculos que conectan páginas web entre sí, permitiendo a los usuarios navegar de una página a otra.
  • Compatibilidad con Navegadores: HTML es un estándar ampliamente aceptado y compatible con la mayoría de los navegadores web, lo que garantiza que las páginas web se visualicen correctamente en diversas plataformas y dispositivos.

Procedimientos

Utilizar HTML implica seguir una serie de procedimientos para crear y estructurar el contenido de una página web.

Los principales pasos son:

  • Crear un Documento HTML: Se abre en editor de texto preferido (como Notepad, Visual Studio Code, Sublime Text).
  • Agregar el Contenido: Utilizar etiquetas HTML como <body> y </body> para el contenido de la página. Añadir enlaces e imágenes usando las etiquetas <a> y <img>.
  • Crear Listas: Usar etiquetas <ul> y <li> para listas no ordenadas, y <ol> y <li> para listas ordenadas.
  • Aplicar Estilos con CSS: Enlazar un archivo de estilo CSS usando la etiqueta <link> en la sección <head>.

Resultados

Se aprendió que las páginas web desarrolladas con HTML tienen una estructura coherente y organizada, facilitando la comprensión y navegación para los usuarios y motores de búsqueda.

Las páginas HTML son compatibles con la gran mayoría de los navegadores web, asegurando que el contenido se vea y funcione de manera consistente en diferentes plataformas y dispositivos.

Aprender a planificar y gestionar proyectos web desde la concepción hasta el despliegue.

Las páginas bien estructuradas en HTML pueden ser más fácilmente indexadas por los motores de búsqueda para mejorar la visibilidad en los resultados de búsqueda y la accesibilidad para los usuarios.
