Semana 3: CSS y Librerías

Definiciones

CSS es un lenguaje de estilo que describe cómo se presenta un documento HTML, donde se definen reglas que especifican el estilo y diseño de los elementos en una página web.

Las reglas CSS consisten en selectores que apuntan a elementos HTML específicos y declaraciones que definen los estilos aplicados a esos elementos.

Las librerías de CSS son conjuntos predefinidos de estilos y componentes que se pueden utilizar en proyectos web para agilizar el desarrollo y mantener una apariencia coherente. Algunas de las librerías más populares son:

  • Bootstrap: Es un marco de diseño (framework) front-end que proporciona una colección de componentes y estilos predefinidos, basados en HTML, CSS y JavaScript. Facilita la creación de interfaces de usuario responsivas y atractivas de manera rápida, sin necesidad de diseñar todo desde cero.
  • Tailwind CSS: Es un enfoque diferente que proporciona clases utilitarias directamente en el HTML para estilizar elementos. Ofrece flexibilidad y control detallado sobre el diseño y los estilos, permitiendo una personalización profunda.

Procedimientos

CSS se utiliza para aplicar estilos como colores, fuentes, márgenes, rellenos y bordes a los elementos HTML, permitiendo una presentación visual atractiva.

Para el uso de CSS y su conexión con HTML y sus librerías, los principales pasos son:

  • Crea un archivo CSS externo: (por ejemplo, estilos.css) y enlázalo con tu documento HTML usando la etiqueta <link> en la sección <head>.
  • Aplicar estilos: Utiliza clases (.) y IDs (#) para aplicar estilos a elementos específicos como color, font-size, margin, padding, border, etc., para controlar la apariencia de los elementos.
  • Bootstrap: Vincula el archivo de estilo de Bootstrap en tu HTML y utiliza las clases predefinidas de la documentación.
  • Tailwind CSS: Instala con npm o utiliza un CDN. Aplica las clases utilitarias directamente en el HTML.

Resultados

El uso de CSS y sus librerías en el desarrollo de páginas web puede generar una serie de resultados beneficiosos que impactan directamente en la apariencia, la interactividad y la eficiencia del desarrollo.

El diseño responsivo de las librerías garantiza una experiencia óptima en dispositivos móviles.

Las librerías como Bootstrap y Tailwind CSS proporcionan componentes y estilos predefinidos que aceleran el desarrollo.
