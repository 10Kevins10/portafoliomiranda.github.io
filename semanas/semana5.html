Semana 5: Desarrollo con React

Definiciones

React es una biblioteca de JavaScript que se utiliza para crear interfaces de usuario, así como para desarrollar aplicaciones de una sola página (SPA).

React se utiliza para una variedad de propósitos en el desarrollo web y móvil, como:

  • Desarrollo de aplicaciones web interactivas, permitiendo la creación de interfaces de usuario dinámicas y altamente interactivas.
  • Facilita la manipulación y modificación de elementos HTML y CSS en tiempo real.
  • Desarrollo de aplicaciones móviles utilizando React Native y su integración con otras bibliotecas y frameworks.

Procedimientos

Trabajar con React implica una serie de procedimientos que abarcan desde la configuración inicial del entorno de desarrollo hasta la creación de componentes y el manejo del estado.

Para trabajar con React, realiza los siguientes pasos:

  1. Instala Node.js y ejecuta el comando: npx create-react-app mi-aplicacion-react.
  2. Explora la estructura de carpetas y archivos generada por Create React App.
  3. Crea componentes de React utilizando funciones y clases, los detalles se explicarán en las siguientes semanas.

Resultados

Se observó que el uso de React en el desarrollo de páginas web puede llevar a varios resultados positivos que benefician tanto a los desarrolladores como a los usuarios:

  • React facilita el mantenimiento del código y acelera el desarrollo al permitir reutilizar componentes en diferentes partes de la aplicación.
  • Facilita el manejo de la lógica de la aplicación y la actualización dinámica de la interfaz de usuario en respuesta a cambios de estado.
  • Cuenta con un amplio ecosistema de bibliotecas y herramientas, como Redux para la gestión del estado, entre muchas otras.
