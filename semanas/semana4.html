Semana 4: JavaScript

Definiciones

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es conocido comúnmente como el lenguaje de script para la web, ya que se utiliza principalmente en el desarrollo de sitios web interactivos y dinámicos.

Es un lenguaje de scripting, lo que significa que se ejecuta directamente en el navegador sin necesidad de compilación.

Dentro de sus funciones y para qué sirve están:

  • Permite crear páginas web interactivas al responder a las acciones del usuario, como clics, desplazamientos y entradas de teclado, validación de formularios en tiempo real, carga dinámica de contenido, efectos de desplazamiento suave, etc.
  • Facilita la manipulación y modificación de elementos HTML y CSS en tiempo real.
  • Ofrece estructuras de control de flujo como bucles y condicionales para controlar el comportamiento del programa y realiza operaciones condicionales.
  • Se utiliza en el desarrollo de aplicaciones web complejas, incluidos marcos y bibliotecas como React, Angular y Vue.js.

Procedimientos

Para utilizar JavaScript en el desarrollo web, hay varios procedimientos básicos, pero los principales son:

  • Se puede incluir el código JavaScript directamente en los archivos HTML mediante la etiqueta <script>.
  • JavaScript se utiliza comúnmente para responder a eventos del usuario. Asocia funciones de JavaScript a eventos HTML utilizando atributos como onclick, onmouseover, document.getElementById, document.querySelector, etc.
  • También se puede declarar variables usando var, let, o const y realizar operaciones básicas como suma, resta, multiplicación, etc.
  • Por último, se puede explorar frameworks y bibliotecas de JavaScript como React, Angular o Vue.js para facilitar el desarrollo de aplicaciones web más complejas.

Resultados

Se observó que el uso de JavaScript en el desarrollo de páginas web puede tener varios resultados beneficiosos que mejoran la interactividad, la dinámica y la experiencia del usuario.

También que JavaScript facilita la manipulación y modificación de elementos HTML y CSS en la página.

JavaScript es esencial para construir aplicaciones web de una sola página que ofrecen una experiencia de usuario más fluida.

JavaScript facilita la integración con APIs y servicios externos, permitiendo la comunicación entre la página web y otros sistemas.

Por último, JavaScript se puede utilizar para desarrollar juegos sencillos y aplicaciones interactivas en el navegador.
