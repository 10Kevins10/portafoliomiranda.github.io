Conclusiones

1. Mostrar habilidades técnicas: La construcción de un portafolio web utilizando HTML y CSS demuestra un dominio sólido de las bases del desarrollo web. La incorporación de JavaScript y React muestra una comprensión avanzada de la interactividad y la creación de aplicaciones web modernas.


2. Demostrar creatividad: El diseño de tu portafolio es una oportunidad para destacar tus habilidades de diseño web. Puedes mostrar tu capacidad para crear interfaces atractivas y amigables para el usuario.


3. Mejorar tu presencia en línea: Un portafolio web bien diseñado puede aumentar tu visibilidad en línea y ayudarte a destacar en un mercado competitivo.


4. React proporciona un enfoque claro para gestionar el estado de la aplicación, lo que facilita la manipulación y actualización de datos de manera eficiente.


5. React y Next.js permiten un desarrollo eficiente y modular mediante la creación de componentes reutilizables. Esto simplifica la gestión del código y facilita la escalabilidad del proyecto.


Reflexión

REFLEXION METACOGNITIVA: La creación de un portafolio que destaque mis habilidades en desarrollo web ha sido un proceso enriquecedor desde una perspectiva metacognitiva. En este proceso, he tenido que reflexionar constantemente sobre mis elecciones, decisiones y desafíos en cada etapa de desarrollo. Algunos de los aspectos más significativos que he observado son: Una planificación estrategica, la eleccion de tecnologias de desarrollo web como (HTML, CSS, JavaScript y React.), un diseño para considerar la experiencia del usuario y evaluar mis conocimientos y progresos


¿QUE APRENDI Y COMÓ? Aprendi la creación de una página web utilizando HTML, CSS, JavaScript y React.; lo cual ha sido una experiencia de aprendizaje continua y gratificante. Al sumergirme en estos lenguajes y tecnologías, he adquirido una comprensión profunda de varios aspectos clave:


Semana 1 y 2: Estructura y marcado con HTML Aprendí a utilizar HTML para crear una estructura sólida para mis páginas web. Descubrí cómo organizar el contenido de manera semántica, tanto la generacipon de codigos emmet para la simplificar las tablas de manera rapida y eficaz.

Semana 3: Diseño con CSS CSS me permitió explorar la creatividad en el diseño web. Aprendí a aplicar estilos a elementos HTML y a diseñar interfaces atractivas y funcionales. Entender la cascada, la herencia y la especificidad de CSS fue fundamental para controlar la apariencia de mis páginas.

Semana 4: Interactividad con JavaScript JavaScript me abrió la puerta a la interactividad y la dinámica en mis proyectos. Aprendí a crear funciones y eventos, lo que me permitió responder a las acciones del usuario y manipular el contenido en tiempo real.

Semana 5: Desarrollo de aplicaciones con React Al utilizar React, comprendí cómo construir aplicaciones web más complejas y dinámicas. Aprendí a dividir la interfaz en componentes reutilizables, lo que facilita la organización y el mantenimiento de proyectos grandes. La gestión de estados y el enrutamiento en React también se convirtieron en habilidades clave.

Semana 6: Desarrollo de aplicaciones con Eventos, Condicionales y Bucles con Framework JS Al aprender a usar Java dentro de React, note el uso tanto de diferentes condicionales y bucles que eston ofrecen. Por ello fue necesario la guia de varios materiales para asi entender gran parte de su programación.

Semana 7: Uso de Hooks Al usar hooks como useState, useEffect, useContext entre otros, aprendi como se separan sus componente tanto en react como next, y que estos al usarse con diferentes librerias, pueden hacer que la interfaz de la pagina web sea de la mejor manera posible.

Semana 8: Evaluación Parcial del Consolidado I Dentro de la evaluación parcial, que fue la creación de una ruleta rusa con diferentes botones, aprendi más de la programación en JavaScript usando Canvas, lo que permite una mejor interfaz e interación para la pagina web.

Semana 9: Next JS CSR / SSR Al adentrarme a usar Next.js pude notar su diferencia con React, lo cual permite comprender mejor como es el uso de las APIs y la separación de carpetas dentro de Next, junto con sus diferentes funcionalidades que este ofrece.

Semana 10: Consumo de APIs dentro de Next.Js Aprendi a usar diferentes APIs como la de "Rick and Morty", "Breaking Bad", "Pokemon" y "Marvel", lo cual es bueno para los trabajos que necesitan consumir estos para el desarrollo de paginas web tanto para Next.js y React.

Semana 11: Next Auth y Server Side Props en Next.js Aprendi las diferentes propiedades que ofrece Next.js, asi como su creación y ejecución dentro de Visual Studio, lo cual srive para una mejor creación para el desarrollo de paginas web.

Semana 12: Desarrollo backend Aprendi a diferenciar a fondo tanto el desarrollo frontend como el backend, en el cual dentro de este último, se consume todo lo que son los archivos Json y las bases de datos de los programas.
